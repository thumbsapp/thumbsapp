# ThumbsApp

ThumbsApp is a competitive social platform combining elements of Facebook, GETTR, Twitch, and TikTok. It features a three-column layout, challenge cards, live matches, ThumbTok short videos, escrow visualization, and a marketplace.

## Deployment
This is a static site, deployable on GitHub Pages. All data is stored in localStorage and JSON files.

## Features
- Feed with challenge cards including reputation, level, country, etc.
- Left sidebar with identity and navigation.
- Right rail with live matches and donation ticker.
- ThumbTok vertical video with scroll play.
- Messenger page with conversations.
- Leaderboard with filters.
- Marketplace with gift cards.
- Escrow transactions visualization.
- Challenge creation modal.
- Global state with localStorage persistence.

Open any HTML file in a browser. For full functionality, serve via a local server (e.g., `python -m http.server`) to load JSON files via fetch.
