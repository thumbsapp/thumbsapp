# ThumbsUp.com

A Gen-Z-oriented social competition platform combining GETTR-style content, Facebook social features, Twitch-style live streams, Roblox-lite avatars, and deterministic 1v1 skill games.

## Features
- Three-column layout with feed, live arenas, and navigation
- Real-time WebSocket updates
- 200+ deterministic skill games
- Escrow system with audit logs
- ThumbsCoins virtual currency
- Marketplace with verified gifts
- Donations and sponsorships
- Avatar customization (Roblox-lite)

## Tech Stack
- Next.js (React + TypeScript)
- Zustand for state management
- Tailwind CSS for styling
- Socket.IO for realtime
- Deterministic game engines

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`

## Deployment
Ready for deployment on Vercel or any Node.js hosting.
